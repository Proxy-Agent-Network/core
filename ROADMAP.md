# Product Roadmap (2026)

This document outlines the development trajectory for the Proxy Agent Network.

## Q1 2026: The "Genesis" Phase (Current)
* [x] **Core Protocol:** Release v1 API for Human-in-the-Loop tasks.
* [x] **Legal Framework:** Standardize "Power of Attorney" specifically for AI Agents.
* [ ] **Lightning Integration:** Finalize micro-payment rails for sub-second settlement.
* [ ] **Python SDK:** Release `pip install proxy-agent` (Alpha).

## Q2 2026: The "Identity" Phase
* **Video Authentication:** Real-time deepfake-resistant video challenges for Agents.
* **Notary API:** Integration with 50-state remote online notarization (RON) providers.
* **Escrow Smart Contracts:** Move from centralized escrow to on-chain DLCs (Discreet Log Contracts).

## Q3 2026: The "Decentralization" Phase
* **Proxy Node Software:** Allow anyone to run a "Proxy Node" and earn fees for verifying tasks.
* **Reputation Protocol:** On-chain reputation scoring for Human Proxies (slashing mechanics for bad actors).
* **Marketplace:** Public order book for complex physical tasks (e.g., "Go to this address and take a photo").

## Q4 2026: The "Governance" Phase
* **DAO Structure:** Community voting on fee parameters and supported jurisdictions.
* **Global Expansion:** EU/GDPR compliance layer.

---
*Note: Timelines are estimates. Priorities may shift based on Agent adoption.*
